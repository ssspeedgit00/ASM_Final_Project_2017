## ASM Fianl Project 2017 - OURCRAFT 0D
**OURCRAFT 0D** , an Assembly(**NASM**) game project with **OpenGL** **GLEW** **GLFW** **glm** on **OS X**.
- dependencies
```
brew install glew glfw glm
```
- Start the game
```
./run.sh
```
Enjoy the game :D
